# Button Loading State Demo

## 📌 Description
This demo shows a button loading state to improve user experience and prevent multiple clicks.

## ✨ Features
- Loading state on button click
- Button disables during processing
- Prevents duplicate submissions
- Smooth UX behavior

## 🚀 How to Run
Open `demo.html` in browser and click the button.
